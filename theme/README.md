# Gatsby Theme Jam

# Installation

To use this theme in your Gatsby sites, follow these instructions:

1. Install the theme

`npm install --save gatsby-theme-jam-example`

2. Add the theme to your `gatsby-config.js`:

```
module.exports = {
  plugins: [
    'gatsby-theme-jam-example'
  ]
}
```

3. Start your site

```
gatsby develop
```
